== Description ==

Infinite Notification is for product sales notification that you need for everyday use in the website to boost your sale.

== Installation ==

This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload `infinite-notification` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start using it by going to the Infinite Notication menu from settings.
